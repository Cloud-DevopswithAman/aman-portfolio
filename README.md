# Aman Kumar Singh — Portfolio

My personal portfolio site — DevOps Engineer & Site Reliability Engineer, showcasing 4+ years of
production infrastructure work across AWS and Azure, plus my current work bringing agentic AI
into Kubernetes operations.

**Live site:**https://cloud-devopswithaman.github.io/aman-portfolio/

---

## What's here

A single-page, static portfolio built with plain HTML/CSS/JS — no framework, no build step.
Designed around a systems-monitoring / status-dashboard aesthetic to match the subject matter:
status indicators, a live terminal-style demo, and log-style experience entries.

**Sections:**
- **Hero** — role, quick stats (years of experience, microservices supported, efficiency gains)
- **About** — background and current focus
- **Signature project** — a live-animated terminal demo of my
  [AI Kubernetes Troubleshooting Agent](https://github.com/Cloud-DevopswithAman/kuberenetes-AI-agent),
  showing real investigation output: root cause, confidence score, suggested fix
- **Experience** — chronological work history at TCS across AWS and Azure
- **Skills** — grouped by category: Cloud, IaC & Pipelines, Containers & CI/CD, Observability &
  Security, AI & Platform Automation, SRE & Operations
- **Certifications** — AWS SAA, AWS CCP, GCP Professional Cloud Architect
- **Contact** — email, phone, LinkedIn, GitHub

## Tech

- HTML5, CSS3 (custom properties, CSS Grid/Flexbox), vanilla JS
- No dependencies, no package manager, no build process — open `index.html` and it runs
- Responsive down to mobile, keyboard-navigable, respects `prefers-reduced-motion`

## Running locally

Clone the repo and open `index.html` directly in a browser:

```bash
git clone https://github.com/Cloud-DevopswithAman/Cloud-DevopswithAman.github.io.git
cd Cloud-DevopswithAman.github.io
open index.html   # or just double-click the file
```

## Deployment

Hosted for free on **GitHub Pages**, served directly from the `main` branch root. Any push to
`main` updates the live site within a couple of minutes.
